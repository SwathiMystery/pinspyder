Pinspyder

Crawl any Pinterest profile page

Download as .zip Download as .tar.gz View on GitHub

Welcome to pinspyder

pinspyder : Crawl any Pinterest profile and fetch the user information, board information, pin information and social information.

Quickstart

$ git clone https://github.com/SwathiMystery/pinspyder.git
$ cd pinspyder/PinterestScraper/
$ mvn clean install

Sample

Input:

http://www.pinterest.com/techcrunch/
